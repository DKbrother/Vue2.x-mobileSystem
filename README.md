# 基于Vue2.0的移动端项目

> 基于Vue.js2.0的移动端商城类APP

### 技术栈
- Vue2.0
- Vue-Router  
- Mint-Ui
- Mui
- Axios

### 使用方法
- 将项目clone或download至本地，执行下面命令
	+ npm install  // 下载相关依赖包
	+ npm run dev  // 开启服务器，即可看到效果页面
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
webpack
```
